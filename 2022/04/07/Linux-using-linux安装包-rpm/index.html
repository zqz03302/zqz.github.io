    Linux_using_linux安装包_rpm

    linux安装包简介

    软件包的安装本质就是软件的安装。软件的安装和运行，其实本质就是程序的运行。你可以选择源码直接编译，也可以用编译以后的二进制包安装。安装包就是编译好的二进制包。

    linux安装包种类

    linux的安装包有rpm和deb的分类，主要面对不同流派的系统。rpm为Redhat的格式，deb为Debain的格式。

    依赖性

    安装的软件需要安装其他的软件，就是依赖性。有时候需要安装某个其他软件的组件，就可以去查询www.rpmfind.net这个网站。

    rpm安装包命令

    rpm 【选项】 包名

    -i 安装

    -v 显示详细信息

    -h 显示进度

    -nodeps 不检测依赖性（不建议使用）

    rpm安装包查询

    rpm 【选项】 包名

    -q 查询

    -a 所有

    -i 信息

    -p 查询未安装包的信息

    -l 位置（列表展示）

    -f 查询文件属于那个包

    -R 查询软件的依赖性

    校验和提取

    rpm 【选项】 包名

    -V 效验安装包

    注意：验证有8个信息。S M 5 D L U G T。依次对应的是，文件大小、权限、内容、代码、路径、所有者、所属组、修改时间。还有5个文件类型，c d g l r，依次为配置文件、普通文档、鬼文件(不应该含有的文档)、授权文件、描述文件。

    提取文件 rpm2cpio 包名 | cpio 【选项】【文件/设备】

    -i 还原

    -d 自动新建目录

    -v 显示还原过程
