    Linux_using_用户相关文件和命令

    家目录

    普通用户：/home/用户名，所有者所属组共用

    root用户：/root，所有者所属组共用

    用户邮箱

    目录：/var/spool/mail/用户名/ （var目录是linux可变数据保存目录）

    用户模板目录

    目录： /etc/skel/

    该目录是创建用户的模板目录，如目录下创建文件A。新用户创建用户后，家目录下就有A文件。

    命令：useradd

    作用：加用户
    -u UID：手工指定UID
    -d 家目录：手工指定家目录
    -c 用户说明：手工指定用户说明
    -g 组名：指定的初始组
    -G 组名：指定用户的附加组
    -s shell：手工指定用户登录的shell，默认是/bin/bash

    默认文件：
    /etc/default/useradd
    GROUP 默认组
    HOME 默认家目录
    INACTIVE 密码过期天数宽限（-1为失效）
    EXPIRE 密码失效时间
    SHELL 默认shell
    SKEL 模板目录
    CREATE_MAIL_SPOOL 是否建立邮箱
    /etc/login.defs
    PASS_MAX_DAYS 密码有效期
    PASS_MIN_DAYS 密码修改间隔
    PASS_MIN_LEN 密码最小位
    PASS_WARN_AGE 密码到期警告
    UID_MIN 最小和最大UID范围
    GID_MAX
    ENCRYPT_METHOD 加密模式

    命令：passwd

    作用：密码操作
    -S 查询密码状态
    -l 暂时锁定用户
    -u 解锁用户
    –stdin 通过管道符输出数据作为用户的密码

    注意：在初始化密码或者修改密码的时候，后退键（backspace）也会被当做字符输入，所以不能按后退键删除字符

    命令：usermod

    作用：修改存在用户
    -u UID：手工指定UID
    -d 家目录：手工指定家目录
    -c 用户说明：手工指定用户说明
    -g 组名：指定的初始组
    -G 组名：指定用户的附加组
    -s shell：手工指定用户登录的shell，默认是/bin/bash

    命令：change

    作用：修改用户密码状态
    -l 列出用户详细密码状态
    -d 日期 修改密码的最后一次更改日期
    -m 天数 两次密码的修改间隔
    -M 天数 密码有效期
    -W 天数 警告天数
    -I 天数 宽限天数
    -E 日期 失效日期
    注意，这个change命令其实就是去修改了shadow下的参数。

    命令：userdel

    作用：删除用户
    -r 删除用户的同时删除用户的家目录

    命令：id

    作用：查看用户的id

    命令：su

    作用：切换用户
    —— ： 加上-的时候表示，代表环境目录都切换了

    -c： 只运行一次，不切换目录

    命令：groupadd

    作用：添加用户组

    -g GID 指定组ID

    命令：groupmod

    作用：修改用户组

    -g GID 修改组ID
    -n 新组名 修改组名

    如把A组改成B组
    groupmod -n A B

    命令：groupdel

    作用：删除组

    注意：其中组中有成员不能直接删除，但是附加用户没有关系

    命令：gpasswd

    作用：把用户添加入组或删除

    -a 用户名 把用户加入组
    -d 用户名 把用户从组中删除

    注意：加入的都是附加组
