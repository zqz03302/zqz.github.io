    三种模式

    vim三种模式

    1.命令模式

    2.插入模式

    3.编辑模式

    ！三种模式切换

    命令模式

    插入：

    a 光标所在字符后插入

    A 光标所在行尾插入

    i 光标所在字符前插入

    I 光标所在行头插入

    o 在光标下插入新行

    O 在光标上插入新行

    set nu 设置行号

    set nonu 取消行号

    gg 到第一行

    G 到最后一行

    nG 到第n行

    ：n 到n行

    $ 到行首

    0 到行尾

    删除：

    x 删除光标后的一个字符

    nx 删除光标后的n个字符

    dd 删除光标所在行

    ndd 删除n行

    dG 删除所在行直到结尾

    D 删除光标处直到结尾

    ：n1,n2d 删除范围行

    复制和剪切：

    yy 复制这一行
    nyy 复制以下的n行
    dd 剪切这行
    ndd 剪切n行
    p、P p是向下黏贴，P是向上黏贴

    替换和取消

    R 进入替换模式直接替换
    u 类似于后退

    搜索：
    /搜索的字符 （n键下翻 N上翻）

    注意：如果想不区分大小写搜索，可以输入：set ic，取消是：set noic

    替换：

    ：替换范围+/要替换的字符串/换成的字符串+g或者c（这个区别在于替换时候询问不询问）

    替换范围格式：

    全文 %s

    指定行数 起始行，终止行s

    如 :50,60s/abc/cba/g 就是在50-60行搜索abc字符修改为cba

    保存退出：

    w 保存

    q 退出

    wq！保存强制退出（！有强制的意思）

    w + 新文件名 相当于另存为

    shift +zz 保存快捷退出
