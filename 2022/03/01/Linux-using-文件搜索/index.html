    Linux_using_文件搜索

    命令：find

    作用：文件搜索 find 【搜索范围】 【匹配条件】

    -name 按照名字搜索 find 【搜索范围】 -name 【名字】

    高级：对于名字来说，单个输入名字是精准匹配。如：find / -name a 就是全盘查找a这个文件。但是要找含有a的，就应该把a改成*a*，*表示代替任何字符串。同时如果你想找的文件叫做axxx（x代表字符，如abcd），就需要把a改成a???。?代表的是一个字符。

    -iname 按照名字搜索，但是不区分大小写。

    -size 根据文件的大小查找。find 【搜索范围】 -size 【文件大小】

    高级：文件的大小假设为n， 【文件大小】+n表示大于这个n的大小 -n表示小于 =n表示等于。其次，linux中一个数据块是0.5k，也就是说，你想查找大于2k的文件，输入的应该是+4，当然现在最方便的是直接写上+2k也是可以的。

    -user 根据文件的所有者查找 。find【搜素范围】 -user 【用户名】

    时间属性

    -amin：根据访问时间

    -cmin：根据更改文件属性时间

    -mmin：根据文件内容改变

    例子：find /etc -mmin -5 ：在etc目录下查找5分钟之内内容改变的文件。

    多条件

    -a：同时满足两个条件

    -o：两个条件满足一个

    例子：find /etc -size +10k -a -size -2000 ：在etc目录下查找一大于10k小于1000k。

    -tpye 根据文件类型

    d是目录

    f是文件

    l是软连接

    -inum 根据节点查找（硬链接）

    高级：在找到的文件以后相对找到的文件直接操作，需要加上-exec 【具体操作】 {} \ 。（其中花括号是找到内容的代称，\是转义字符）

    例子：find /etc -name abc* -exec ls -l {} \ ：表示找到在etc目录下，名字带有abc开头的文件，然后查看详细信息

    命令：locate

    作用：在文件资料库搜索文件 locate 【文件名】

    注意：这个并不是实时性的查找，只是在收录范围查找。新创建的文件并不是实时收录，如果需要可用updatedb这个命令手动更新。其次在/tmp目录下是不收录的。

    命令：which

    作用：搜索命令所在目录的目录和信息。which 【命令】

    注意：usr/bin 是所有用户可使用命令 usr/sbin 就是root使用的命令

    命令：whereis

    作用：搜索命令所在目录和帮助信息。 whereis 【命令】

    命令：grep

    作用:文件搜索命令。grep -iv 【指定字符串】【文件】

    -i 不区分大小写

    -v 排除指定字符串

    例子：这个命令是查找在文件中的字符串。如：grep -i abc /etc/sources.list 在sources.list这个文件中查找带有abc的那一行，而且不区分大小写（也就是Abc也能查找到）。
