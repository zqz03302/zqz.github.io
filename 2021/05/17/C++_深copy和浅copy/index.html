c++——派生类

派生类类的产生就是继承最好体现。简单打个比方来说：
你手头有一个基本的类（基类），但是你对这个基类不是很满意，希望改造一下这个基类。那么最简单的就是使用继承，新建一个类（派生类），可以继承原有基类的所有数据成员和函数（构造和析构除外），并且自己还可以数据成员和函数。
-语法

 class 派生类名：继承方式 基类名 1,继承方式 基类名2,.......
 {
     派生类成员申明；
 };

一个派生类可以继承好几个基类（多继承），可以单继承（单继承）。

而且，继承了基类的派生类又可以作为其他类的基类，就像家族树一样（你爸爸是你爷爷的儿子，你是你爸的儿子）。所以整个就是一个类族。

在类族中，直接派生的就是直接基类（就像家族中的直接血亲）。不是直接派生的，也就是隔了两代以上的就是间接基类。

继承还要定义清楚继承方式：public protected和private三种。如果不写就是private

1.公有继承

基类的私有成员不可以直接继承，但是公有和保护可以直接继承（原有类型不变，公有继承以后为公有，保护继承以后为保护）

2.私有继承

基类的私有成员不可以直接继承，但是公有和保护可以直接继承，但是继承以后公有和保护都变成私有。

3.保护继承

基类的私有成员不可以直接继承，但是公有和保护可以直接继承，但是继承以后公有和保护都变成保护。
