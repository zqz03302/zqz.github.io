c++——函数指针和对象指针

1.函数指针

-语法

数据类型 ( * 函数指针名)(形参表)

如：
int * intPart(int i)

这个就是命名了一个intPart的函数指针。同时这个指针可以接受函数的赋值，或者说是函数的入口(这个指针其实指向的就是函数的入口地址)。

2.对象指针

-语法

类名 * 对象指针名；

-this 指针

是一个特殊指针，指向是现在被成员函数操作的对象。简单来说就是一个类，可能同时产生不同的对象。防止成员函数弄错引用的变量，就会引用this指针。

假设：

在上面A中，有一个X变量，而它的成员中还有一个B，B中还有一个X。那么happy()函数调用哪个X呢？
凭直觉肯定是A，但是为什么？
如果我们把return x; 改为 this->x;而这个this指的就是A这个对象就清楚多了。平常我们使用的时候就是这个this指针是隐藏的。

-指向类的非静态成员的指针

-语法

类型说明符 类名::* 指针名； //申明指向数据成员的指针
类型说明符 (类名::* 指针名)(参数表)； //申明指向函数成员的指针

申明完了以后还要赋值，也就是指明该指针指向那个成员。（不然就是野指针）

指针名=&类名::数据成员名;
